## Mettre le site en ligne avec GitHub Pages (gratuit)

1. Crée un nouveau dépôt sur GitHub (par exemple `portfolio`).
2. Importe **tous les fichiers de ce dossier** à la racine du dépôt (glisser-déposer sur la page GitHub, ou via `git add` / `git commit` / `git push`).
3. Dans le dépôt, va dans **Settings → Pages**.
4. Sous "Build and deployment", choisis **Source : Deploy from a branch**.
5. Choisis la branche `main` et le dossier `/ (root)`, puis **Save**.
6. Après une minute ou deux, ton site sera visible à l'adresse indiquée en haut de cette page (du type `https://ton-pseudo.github.io/portfolio/`).

## Important
- Le fichier principal s'appelle **index.html** — ne le renomme pas, c'est ce nom précis que les hébergeurs (GitHub Pages, Netlify, etc.) affichent automatiquement.
- Toutes les images et le son doivent rester **dans le même dossier** que `index.html` (ne pas les mettre dans des sous-dossiers), car le code fait référence à elles par leur nom simple (ex: `ge-couverture.jpg`).
- Pour modifier un texte ou une image plus tard, tu peux éditer `index.html` directement depuis l'interface GitHub (bouton crayon ✏️ sur le fichier).
